# Mission Boards - Student Portal 📚

A dynamic study management and tracking platform built to help students prepare for their board examinations efficiently.

## 🚀 Live Demo
[Click here to view live project](https://missionboards.netlify.app/)

## 💡 Key Features
* **Secure Authentication:** User login and registration system.
* **Daily Streak Tracker:** Gamified tracking system to maintain study consistency.
* **Issue Reporting:** Automated system for students to report issues or ask queries.
* **Interactive UI:** Smooth transitions and accessible design.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend (BaaS):** Firebase Auth
* **Deployment:** Netlify
